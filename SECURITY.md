# Security Policy

## Supported Versions

The latest published `0.x` release receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report it privately via
[GitHub Security Advisories](https://github.com/guilhermeseckert/tanstack-query-keys/security/advisories/new)
or by email to **guilherme-sonnemann@hotmail.com**.

You can expect an acknowledgement within 72 hours. Once the issue is confirmed
and fixed, a patched release will be published and the reporter credited (unless
anonymity is requested).
